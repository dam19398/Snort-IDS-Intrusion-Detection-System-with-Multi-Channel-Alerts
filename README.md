## 📋 Project Overview

This project implements a **Network Intrusion Detection System (IDS)** based on **Snort 2.9.15.1**, deployed in a virtualized environment using two machines:

- 🖥️ **Ubuntu Server** — Target machine running Snort IDS
- 💀 **Kali Linux** — Attacker machine used to simulate network intrusions

Alerts generated by Snort are delivered through **multiple channels**: Email (Postfix/Gmail), Slack, and Grafana.

## 📦 Installation & Configuration

### Prerequisites

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install -y build-essential libpcap-dev libpcre3-dev \
libdumbnet-dev bison flex zlib1g-dev liblzma-dev openssl \
libssl-dev git curl wget python3 python3-pip
```

### 1️⃣ Install DAQ

```bash
cd /tmp
wget https://www.snort.org/downloads/snort/daq-2.0.7.tar.gz
tar -xvzf daq-2.0.7.tar.gz && cd daq-2.0.7
./configure && make && sudo make install && sudo ldconfig
```

### 2️⃣ Install Snort

```bash
cd /tmp
wget https://www.snort.org/downloads/snort/snort-2.9.15.1.tar.gz
tar -xvzf snort-2.9.15.1.tar.gz && cd snort-2.9.15.1
./configure --enable-sourcefire
make && sudo make install && sudo ldconfig

# Verify installation
snort -V
```

### 3️⃣ Configure Snort

```bash
sudo mkdir -p /etc/snort/rules /var/log/snort
sudo cp /tmp/snort-2.9.15.1/etc/*.conf /etc/snort/
sudo touch /etc/snort/rules/local.rules
sudo touch /etc/snort/rules/white_list.rules
sudo touch /etc/snort/rules/black_list.rules
```

Edit `/etc/snort/snort.conf`:
```bash
ipvar HOME_NET <YOUR_NETWORK>/24
ipvar EXTERNAL_NET !$HOME_NET
var RULE_PATH /etc/snort/rules
include $RULE_PATH/local.rules
```

### 4️⃣ Detection Rules

```bash
# Detect ICMP Ping
alert icmp any any -> $HOME_NET any (msg:"ICMP Ping Detected"; sid:1000001; rev:1;)

# Detect Port Scanning
alert tcp any any -> $HOME_NET any (msg:"Port Scan Detected"; flags:S; \
threshold:type threshold, track by_src, count 5, seconds 10; sid:1000002; rev:1;)

# Detect SSH Brute Force
alert tcp any any -> $HOME_NET 22 (msg:"SSH Connection Attempt"; flags:S; sid:1000003; rev:1;)

# Detect HTTP Traffic
alert tcp any any -> $HOME_NET 80 (msg:"HTTP Traffic Detected"; sid:1000004; rev:1;)

# Detect FTP Attack
alert tcp any any -> $HOME_NET 21 (msg:"FTP Connection Attempt"; flags:S; sid:1000005; rev:1;)
```

### 5️⃣ Slack Alerts

```bash
SLACK_WEBHOOK="https://hooks.slack.com/services/YOUR/WEBHOOK/URL"

curl -X POST -H 'Content-type: application/json' \
--data '{"text":"🚨 *Alerte Snort* 🚨\nSNORT ALERT detected!"}' \
$SLACK_WEBHOOK
```

### 6️⃣ Email Alerts (Postfix + Gmail)

```bash
sudo apt install postfix mailutils -y

echo "Alert Snort Detect.\nDate: $(date)\nAlerte: Test intrusion\nServeur: ubuntu" | \
mail -s "Alerte IDS Snort" your-email@gmail.com
```

### 7️⃣ Grafana Dashboard

```bash
sudo apt install grafana -y
sudo systemctl enable grafana-server
sudo systemctl start grafana-server
# Access: http://<YOUR_SERVER_IP>:3000
```

### 8️⃣ Run Snort as Service

```bash
sudo systemctl enable snort
sudo systemctl start snort
sudo systemctl status snort
```
