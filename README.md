# Intrusion Detection System with Email and SMS Alerting

## Project Overview

This project demonstrates the deployment of a network-based Intrusion Detection System (IDS) capable of monitoring network traffic, detecting suspicious activities, and automatically notifying administrators through email and SMS alerts.

The goal is to improve incident detection and response time by providing real-time security notifications.

## Technologies Used

* Snort
* Linux (Ubuntu/ kali)
* Wireshark
* Python
* SMTP Email Service
* SMS Notification Gateway
* VirtualBox / VMware

## Architecture

Internet Traffic
↓
Network Interface
↓
Intrusion Detection System (Snort)
↓
Alert Generation
↓
Email Notification
↓
SMS Notification
↓
Security Administrator

## Features

* Real-time network monitoring
* Detection of suspicious traffic patterns
* Automated alert generation
* Email notifications for security incidents
* SMS notifications for critical events
* Event logging and analysis

## Implementation Steps

1. Deploy Linux server
2. Install and configure IDS
3. Configure detection rules
4. Create email alerting mechanism
5. Integrate SMS notification service
6. Generate test attacks
7. Validate alert delivery
8. Document findings
